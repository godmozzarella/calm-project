# Calm — сервис по отслеживанию головных болей

Трекер головной боли: дневник приступов и препаратов, аналитика, экспорт для врача.

Монорепо из двух частей:

```
calm-project/
├── frontend/   React 19 + Vite 7 + SCSS modules (FSD-подобная архитектура)
└── backend/    Spring Boot 3.4 + Spring Data MongoDB + JWT
```

## Быстрый старт

### Frontend (dev)

```sh
cd frontend
npm install
npm run dev
```

Откроется `http://localhost:5173`.

### Backend (dev)

Нужна Mongo. Поднять локально:

```sh
docker run -d --name calm-mongo -p 27017:27017 mongo:7
```

Затем:

```sh
cd backend
mvn spring-boot:run
```

API будет на `http://localhost:8080/api`. Подробности — в [`backend/README.md`](backend/README.md).

### Docker (всё сразу)

```sh
docker compose up --build -d
```

Приложение доступно на `http://localhost` (или по LAN IP).

## Стек

- **Фронт:** React 19, Vite 7, SCSS modules, react-router-dom 7, MUI icons. Чистый JS (jsconfig).
- **Бэк:** Java 21, Spring Boot 3.4, MongoDB, jjwt, bcrypt.
- **Хранилище:** MongoDB.

## О проекте

Головная боль является одной из наиболее распространённых проблем со здоровьем, с которой сталкиваются люди разных возрастных групп. При этом многие пользователи не ведут систематический учёт приступов, симптомов и принимаемых лекарственных препаратов, что затрудняет анализ состояния здоровья и выявление факторов, провоцирующих возникновение боли.

Calm позволяет пользователю фиксировать информацию о приступах, указывать принимаемые препараты, сохранять сопутствующие симптомы и возможные триггеры. На основе введённых данных система формирует историю наблюдений и предоставляет аналитику.

**Calm — Отслеживай. Понимай. Живи легче!**

### Проблема

Многие люди сталкиваются с регулярными приступами головной боли или мигрени, однако не всегда ведут систематический учёт симптомов, факторов возникновения и принимаемых лекарств. Отсутствие удобного инструмента приводит к тому, что пользователи не могут отслеживать динамику своего состояния, выявлять возможные причины возникновения боли и оценивать эффективность принимаемых препаратов.

**Воздействует на:**
- людей, страдающих регулярными головными болями или мигренью
- пациентов, проходящих лечение у неврологов
- пользователей, следящих за своим состоянием здоровья
- медицинских специалистов, анализирующих состояние пациентов

**Что даёт система:**
- фиксировать информацию о приступах головной боли
- сохранять данные о симптомах и принимаемых лекарствах
- отслеживать частоту и интенсивность приступов
- анализировать возможные причины возникновения головной боли
- формировать историю наблюдений за состоянием здоровья
- экспортировать отчёт для врача (PDF, CSV)
